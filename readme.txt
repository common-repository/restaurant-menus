== Description ==


An easy way to place a restaurant menu with content on your Wordpress site.

###Quick Start Guide

* [restaurant menu quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/restaurant-menu/)

###What you get when you use the Restaurant Menu restaurant menu


*  Create custom restaurant menu pages.
*  restaurant menu background color
*  restaurant menu description
*  Unlimited amount of restaurant menus
*  Unlimited amount of items on a restaurant menu
*  Restaurant menu photo


###Restaurant Menu 

Add as many restaurant menus to your site as you like.  The Restaurant Menu lets you create restaurant menu and save them for when ever you are ready to use them.



== Installation ==


1. Upload `restaurant-menu` to the `/wp-content/restaurant-menu/` directory
1. Activate the restaurant menu through the 'Restaurant Menu' menu in WordPress
1. Click **Restaurant Menu** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The restaurant menu does the programming and design.

= Is there a limit to the number of restaurant menu I can create? =
No, unlimited restaurant menu can be created.
